= Part 1: Setup & Visual Exploration with Neo4j

This part introduces the structured knowledge graph — you'll load it, explore it, and query it with a no-code agent.

You will sign in to AWS, provision a Neo4j Aura instance, load the SEC 10-K financial dataset, and explore it using graph visualization and natural language queries. By the end, you will have a running knowledge graph and a no-code AI agent built with Neo4j Aura Agents.
